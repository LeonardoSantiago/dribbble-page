# [Projeto Front End]

## Instalação e Uso

+ Instalar todas as dependências.
>
```bash
$ npm install
```
+ Para rodar o server
>
```bash
$ yarn server
```
+ Para rodar os testes(rode primeiro o server e depois o comando de testes)
>
+Primeiro:
```bash
$ yarn server
```
Depois:
```bash
$ yarn test
```
